# mtk-macleaners

Premium cleaning service platform built with HTML, SCSS, Vanilla JavaScript, wc events, wc-include, Node.js, and SQLite.

## Demo
Open `index.html` through a local web server so `wc-include` can fetch `mtk-macleaners.html`.

## Backend
```bash
cd server
npm install
npm start
```
